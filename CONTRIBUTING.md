# Contributing

Thanks for your interest in this project! Contributions are welcome.

## Bug Reports

Open an issue with:

- Firmware version (git tag or commit hash)
- What you expected to happen
- What actually happened
- Steps to reproduce
- Hardware setup (Pico variant, wiring, controller model)

## Pull Requests

- Keep changes focused — one feature or fix per PR
- Follow the existing code style (C11, `-Wall -Wextra`, 4-space indent)
- Test on physical hardware if possible
- If adding a new feature, update the README

## Testing

This project needs physical hardware testing. If you have a Pico and a DS3 controller, see the [Testing Checklist](README.md#testing-checklist) in the README and report results in an issue.
